# Extract log-likelihood

Extract log-likelihood

## Usage

``` r
# S3 method for class 'betaregscale'
logLik(object, ...)
```

## Arguments

- object:

  A fitted `"betaregscale"` object.

- ...:

  Ignored.

## Value

An object of class `"logLik"` with attributes `df` (number of estimated
parameters) and `nobs` (number of observations).
