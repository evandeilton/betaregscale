# Extract fitted values

Extract fitted values

## Usage

``` r
# S3 method for class 'betaregscale'
fitted(object, type = c("mu", "phi"), ...)
```

## Arguments

- object:

  A fitted `"betaregscale"` object.

- type:

  Character: `"mu"` (default) or `"phi"`.

- ...:

  Ignored.

## Value

Numeric vector of fitted values.
